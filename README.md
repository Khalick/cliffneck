## Deployment

### Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/cliffneck)

Or manually:
1. Push your code to GitHub
2. Visit [vercel.com](https://vercel.com)
3. Import your GitHub repository
4. Click "Deploy"

### Deploy to GitHub Pages

1. Go to your repository settings
2. Navigate to "Pages"
3. Select "main" branch as source
4. Click "Save"
